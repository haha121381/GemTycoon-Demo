# **📖 宝石大亨游戏百科 (v1.6)**

## **🃏 卡牌 (Symbols)**

卡牌是构筑卡组的基础单位，每次旋转生成 9 张。

| 名称 | 价格 (G) | 基础效果 | 特殊能力 |

| **蓝宝石** (Sapphire) | 11 | 10 筹码 | 可配合神器\[蓝绿色盲\]连携 |
| **祖母绿** (Emerald) | 13 | 15 筹码 | 可配合神器\[蓝绿色盲\]连携 |
| **红宝石** (Ruby) | 16 | 20 筹码 | 触发神器\[红运当头\]成长 |
| **金刚石** (Diamond) | 19 | 20 筹码, +2 倍率 | 提供 **+2 倍率** (Add Mult) |
| **塔菲石** (Pink Diamond) | 14 | - | 提供 **+5 倍率** (Add Mult) |

## **🎰 排列系统 (Synergy Patterns)**

旋转后 9 张卡牌按 3x3 网格排列，满足特定形状且同色即可触发排列奖励。

### **排列类型**

| 排列名称 | 形状 | 格子位置 | 基础倍率 |
| :---- | :---- | :---- | :---- |
| **横向三连** | 横排 3 连 | [0,1,2] / [3,4,5] / [6,7,8] | +7 |
| **纵向同花** | 竖排 3 连 | [0,3,6] / [1,4,7] / [2,5,8] | +7 |
| **对角连击** | 对角线 3 连 | [0,4,8] / [2,4,6] | +8 |
| **四方连结** | 2x2 方块 | [0,1,3,4] / [1,2,4,5] / [3,4,6,7] / [4,5,7,8] | +10 |
| **四角同辉** | 四角 | [0,2,6,8] | +15 |
| **缺心十字** | 十字 (缺中心) | [1,3,5,7] | +15 |

### **连携规则**

- 所有参与排列的卡牌必须是**宝石类型**（原石和金币卡不算，除非持有\[老眼昏花\]）
- **彩虹石**可替代任意宝石参与连携
- **蓝宝石 + 祖母绿**在持有\[蓝绿色盲\]时视为同色
- 同一排列可同时触发多个（如横向三连和纵向同花重叠）
- \[眼花缭乱\]使每个排列额外再叠加一次其倍率
- \[缺心眼儿\]仅对"缺心十字"额外 +55x 倍率

## **⚙️ 计分公式**

```
总分 = 筹码总数 × 倍率总数
```

### **筹码来源**

- 卡牌基础筹码
- \[抛光布\]对所有宝石 +10 筹码
- \[地质锤\]对原石 +10 筹码
- \[贪婪之手\]对金币卡 +10 筹码
- \[魔力宝匣\]卡组每张牌 +2 倍率（注意：实际效果是倍率）
- \[矿石标本\]每张原石 +15 筹码
- \[多多益善\]每个神器 +25 筹码
- \[红运当头\]成长值叠加筹码

### **倍率来源**

- 卡牌基础倍率（金刚石 +2、塔菲石 +5）
- \[角磨机\]对所有宝石 +1 倍率
- \[金光闪闪\]每个金币卡 +0.8x 倍率
- \[贤者之石\]每个原石 +3x 倍率
- \[红运当头\]成长值叠加倍率
- \[聚焦透镜\]每次旋转成长 +2 倍率

### **倍乘因子 (乘算)**

- 紫水晶 **x2**（持有\[紫气东来\]时为 **x3**）
- 排列奖励（叠加到倍率上）
- \[完美主义\]总分 **x1.5**（卡组无原石时）
- \[粉红之约\]总分 **x2**（场上有塔菲石时）

### **金币产出**

- 金币卡每张 +3 金币
- 托帕石每张 +1 金币

## **🎮 游戏流程**

### **回合结构**

1. 每个关卡 (Level) 包含 **4 个回合** (Round)
2. 每回合有 **4 次旋转机会**（可通过\[时光沙漏\]增加）
3. 达到目标分数后进入商店
4. 旋转次数用尽且未达标 → 游戏结束

### **目标分数**

| 关卡 | Round 1 | Round 2 | Round 3 | Round 4 |
| :---- | :---- | :---- | :---- | :---- |
| Level 1 | 200 | 250 | 350 | 500 |
| Level 2+ | 递增公式：`目标分 × 1.2 + 200` | | | |
| Level 6+ | 递增公式：`目标分 × 1.25 + 100` | | | |
| Level 10+ | 递增公式：`目标分 × 1.4 + 50` | | | |

> 通关后溢出分数**不会**带入下一回合

### **商店机制**

- 达标后进入商店，可购买卡牌、神器、魔法物品
- 默认 **3 个商品位**（持有\[黑金VIP\]时为 4 个）
- 刷新费用默认 **5G**（持有\[紫金VIP\]时为 2G）
- 刷新次数默认 **1 次**（持有\[货比三家\]时为 3 次）
- \[白金VIP\]全场商品 -5G（最低 1G）
- \[炫彩VIP\]每次刷新随机 1 个商品免费
- \[猪猪存钱罐\]退出商店时，每持有 10 金币获得 1 金币利息
